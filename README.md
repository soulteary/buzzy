## Running your own Buzzy instance

If you want to run your own Buzzy instance, but don't need to change its code, you can use our pre-built Docker image.
You'll need access to a server on which you can run Docker, and you'll need to configure some options to customize your installation.

You can find the details of how to do a Docker-based deployment, or deploy with Kamal for more flexibility (including code changes), in our [deployment guide](docs/deployment.md).
